## 2. 技术格式约束

### 剧情触发器（何时触发）

当前支持的触发器：

| 触发器 | 含义 | 当前使用 |
|--------|------|----------|
| `chapter_start` | 进入该章节时 | Ch1 |
| `boss_pre` | BOSS 战之前 | Ch3, Ch15 |
| `boss_post` | BOSS 战之后 | Ch10 |
| `first_clear` | 章节首次通关 | Ch5 |
| `perfect_clear` | 完美通关 | Ch15（结局） |
| `hidden_event` | 隐藏事件 | Ch7 |

**需求扩展**：为实现"每关都有剧情"，需新增以下触发器（技术侧需同步修改 `story.ts` 类型和调用逻辑）：

| 新增触发器 | 含义 | 频率 |
|-----------|------|------|
| `level_1_clear` | 第 1 关通关后 | 每章 ×1 |
| `level_2_clear` | 第 2 关通关后 | 每章 ×1 |
| `level_3_clear` | 第 3 关通关后 | 每章 ×1 |
| `level_4_clear` | 第 4 关通关后 | 每章 ×1 |

### StoryBeat 数据模型

```
StoryBeat {
  id: string          // 唯一标识，如 "ch1_lv1"
  chapter: number     // 1-15
  trigger: string     // 触发器类型
  format: 'comic' | 'dialogue' | 'narration'
  panels: StoryPanel[]
  rewards: StoryReward[]
  unlockFlags?: string[]
}
```

### 面板类型（StoryPanel）

有三种面板类型，可自由组合：

| 面板类型 | 用途 | 包含字段 |
|---------|------|---------|
| `image` | 插图 + 底部文字说明 | `imagePath`, `text`, `duration` |
| `text` | 角色对话或旁白 | `character`, `emotion`, `text`, `duration` |
| `choice` | 玩家选择分支 | `text`, `choices[]`（含 `text`, `nextPanelId`, `setFlag`） |

### 可用角色情感（emotion）

`smile`😊 / `serious`😐 / `angry`😠 / `sad`😢 / `worried`😟 / `laugh`😈 / `proud`😤 / `cry`😭 / `surprised`😲 / `amazed`🤩 / `warm`🥰 / `epic`✨ / `info`ℹ️

### 插图路径规则

插图存放于 `public/assets/images/story/` 目录，路径命名示例：
- `story/ch1_open_01.png` — Ch1 开场图 1
- `story/ch1_lv1_clear.png` — Ch1 第 1 关通关图

推荐尺寸：**16:9 横版**，建议分辨率 1920×1080。

### 奖励类型

| 类型 | 含义 | 示例 |
|------|------|------|
| `xp` | 经验值 | `{ type: 'xp', amount: 50 }` |
| `gold` | 金币 | `{ type: 'gold', amount: 30 }` |
| `cosmetic` | 收藏品/CG | `{ type: 'cosmetic', id: 'xxx', amount: 1 }` |
| `galleryEntry` | 画廊解锁 | `{ type: 'galleryEntry', id: 'xxx', amount: 1 }` |

## 3. 剧情总览

### 三幕式结构

```
第一幕（Ch1-5）：觉醒与追寻
  └─ 苏醒→得知词根失窃→踏上旅途→获得符文钥匙

第二幕（Ch6-10）：真相与阴影
  └─ 深入黑暗→发现幕后黑手→暗黑骑士揭露远古红龙

第三幕（Ch11-15）：决战与结局
  └─ 集结力量→闯入龙巢→远古红龙之战→双结局
```

### 每章剧情节奏模板

每章 5 关的剧情节奏建议如下：

| 时机 | 篇幅 | 内容定位 |
|------|------|---------|
| **进入章节**（`chapter_start`） | 2-4 面板 | 章节场景描写、环境烘托、目标提示 |
| **第 1 关→第 2 关**（`level_1_clear`） | 1-2 面板 | 简短对话/旁白，怪物残响、道路指引 |
| **第 2 关→第 3 关**（`level_2_clear`） | 1-2 面板 | 角色互动、世界信息碎片 |
| **第 3 关→第 4 关**（`level_3_clear`） | 1-2 面板 | 氛围铺垫、推进主线伏笔 |
| **第 4 关→BOSS**（`boss_pre`） | 2-4 面板 | BOSS 登场/背景介绍、决战动员 |
| **BOSS 通关**（`first_clear` / `boss_post`） | 2-4 面板 | 胜利后剧情推进、关键信息揭露 |

> 每章约需 **5-7 段剧情**，总计约 **85-105 段剧情**。
> 其中 `chapter_start`、`boss_pre`、`boss_post/first_clear` 为核心剧情（较长），
> `level_1_clear` ~ `level_4_clear` 为过渡剧情（简短）。
